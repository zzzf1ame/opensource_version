# 贡献指南 (Contributing Guide)

感谢您对 SmartDoc AI 开源项目的关注！

## 📋 贡献方式

我们欢迎以下类型的贡献：
- 🐛 Bug 报告
- 💡 功能建议
- 📝 文档改进
- 🔧 代码优化

## 🚀 提交 Issue

在提交 Issue 之前，请：
1. 搜索现有 Issues，避免重复
2. 使用清晰的标题描述问题
3. 提供详细的复现步骤
4. 附上相关的日志或截图

## 🔧 提交 Pull Request

1. Fork 本仓库
2. 创建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request

## 📝 代码规范

- 遵循 PEP 8 Python 代码规范
- 添加必要的注释和文档字符串
- 确保代码通过基本测试
